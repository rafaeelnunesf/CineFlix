## Cineflix
This project is about a movie theater seat reservation application

## Why?
My main focus when developing this code was to practice the following subjects:
- React
- Styled Components

I was interested in practice more about how the react `useState` hook works and making changes from those states through styled components props

## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the front-end with
```bash
npm start
```
4. You can optionally build the project running
```bash
npm run build
```
5. Finally access http://localhost:3000 on your favorite browser (unless it is Internet Explorer. In this case, review your life decisions)
